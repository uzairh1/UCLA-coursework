The Assignment

This assignment will give you practice with loops. Here are the rules you must follow:
  1. You may not use either recursion or functions that take other functions as arguments (such as map and filter). Instead, your functions should only use loops.

  2. Do not use any function from Python libraries. Do not use any built-in functions of Python that have not been discussed in lecture. Do not use any language features of Python that have not been discussed in lecture. If in doubt, ask!

The file hw6.py is empty. You should write your functions there. Make sure to use exactly the same function names as specified below, or you will get no credit.
  1. Implement a function halveEvens, which takes a list l of integers and returns a list that is identical to l except that all odd integers have been removed and all even integers have been halved. For example, halveEvens([10,21,32,42,55]) returns [5, 16, 21].

  2. Implement a function splitEveryOther that takes a list l and returns a list of two lists that together contain the elements of the original list in an "every other" fashion. For example, splitEveryOther([1, 4, 23, 3, 8]) returns [[1, 23, 8], [4, 3]]. Important Rule: You are not allowed to use the slicing operation on lists.

  3. Implement a function isSorted, which takes a list of integers and returns True if the list is in sorted order from least to greatest and False otherwise. Duplicate elements are allowed. For example, isSorted([1, 3, 3, 7]) returns True while isSorted([1, 3, 7, 3]) returns False.

  4. In mathematics, the dot product of two vectors of numbers <a1,a2,...,aN> and <b1,b2,...,bN> is defined as a1*b1 + a2*b2 + ... + aN*bN. Implement a function dotProduct that computes the dot product of two vectors, where Python lists are used to represent the vectors. For example, dotProduct([1,2,3], [4,5,6]) returns 32 (since 1*4 + 2*5 + 3*6 = 32). You may assume that the two argument lists have the same length.

  5. Re-implement the negate function from Homework 5, which produces the color negative of an image. But now instead of a map within a map, you'll need to use a loop within a loop! You can assume the same representation of images that we used in that homework: a list of lists of dictionaries. Important Rule: The original list of lists of dictionaries should not be modified at all!

  6. Implement a function toDigitList that takes a positive integer n and returns a list of its digits, in order. For example, toDigitList(403) returns [4, 0, 3]. Important Rule: You may not convert any numbers to strings, or use any string operations of Python. Hint: Iteratively determine the digits from right to left in the number. The modulo operator % and integer division operator // will come in handy.

  7. Consider the process of taking a positive integer n, summing its digits, then summing the digits of the number derived from it, etc., until the remaining number has only one digit. The digit obtained is called the digital root of n, and the number of sums required to obtain a single digit from a number n is called the additive persistence of n.

    For example, 9879 has a digital root of 6 since 9+8+7+9 = 33 and 3+3 = 6. Since two sets of digits were summed in this process, the additive persistence of 9879 is 2.

    Implement a function digitalRootAndPersistence that takes a positive integer n and returns both its digital root and its additive persistence, represented as a dictionary with keys "root" and "persistence". For example, digitalRootAndPersistence(9879) returns {"root": 6, "persistence": 2}. Note: You may use Python's built-in sum function, which sums the elements of a list of numbers, and the toDigitList function that you defined above will also be useful.
