Problem #1: Miscellaneous Functions

  1. Implement the function fizzBuzz, which represents a player for the game fizz buzz. Given a positive integer n, the function should return a string. Specifically, fizzBuzz(n) returns 'fizz' if n is divisible by 3, 'buzz' if n is divisible by 5, 'fizz buzz' if n is divisible by both 3 and 5, and otherwise the function returns the number n itself, but as a string. Note: You can use Python's str function to convert a number into a string. Make sure to return exactly these strings; for example 'Fizz' is not the same as 'fizz'!

  2. On the first day of class we implemented a function that takes the year of someone's birth and returns their age. In this problem you will implement a more specific version of that. Your function, called age, should take three arguments (in this order): the day, month, and year of someone's birth, all represented as integers. The function should then return the age (an integer) that this person will be on the date that the homework assignment is due, October 17, 2024. For example, age(22, 9, 2005) returns 19 but age(22, 10, 2005) returns 18.

  3. Implement a function countPos that accepts an integer list l of any length and returns the number of elements of the list that are positive. For example, countPos([1, -4, 0, 4, 8, 0]) returns 3.

  4. Implement a function threshold that takes a list l of numbers and a number n and returns a new list that is identical to l but with all numbers that are less than n replaced by n. For example, threshold([1, -4, 0, 4, 8], 1) returns [1, 1, 1, 4, 8]. As in the previous problem, the given list can be of any length.

  5. Implement a function intRange that takes two integers low and high and returns an ordered list of all integers that are greater than or equal to low and less than high. For example, intRange(2, 5) returns [2,3,4] and intRange(3, 2) returns []. You may not use the built-in range function.

  6. Implement a function innerLists that takes a list l of integers and returns a list of lists, where each element e of the original list l is replaced by the the list [1,...,e]. For example, innerLists([2, 5, 0, 1]) returns [[1,2], [1,2,3,4,5], [], [1]]. Hint: Make use of the intRange function that you defined in the previous problem. Again, you may not use the built-in range function.

Problem #2: Recursive Art

Turtle graphics is a simple, fun, and powerful way to write programs that draw interesting things. A program simply commands a turtle to move around a canvas, and the path of the turtle determines what is drawn. Python has a library called turtle that supports turtle graphics. You can see its documentation here. I've already imported this library for you at the top of hw2.py.

We will get some practice with the turtle library in class on Monday, October 14. There are lots of functions in the turtle library, but we will only be using two of them. The function turtle.forward(d) moves the turtle forward for distance d (a number), and the function turtle.left(a) turns the turtle left by a degrees. You may not use any other functions from the turtle library!

For this assignment you will implement three functions that use the turtle library to draw things. Note: the turtle functions are unusual in that they don't actually return anything; rather, they just have the effect of drawing on a canvas. Your functions will also not return anything, so our usual recursion recipe doesn't quite apply. Instead, in the recursive case of your functions you'll typically want to first use the turtle functions to draw on the canvas and then make a recursive call in order to draw the rest of the picture.

A few more notes:

  • Though your functions don't return anything, they should not run forever. That is, when one of your functions is called, eventually the turtle should stop moving and be done drawing the picture.
  • If you want to end a function without returning anything, just use the statement return without providing any value to return.
